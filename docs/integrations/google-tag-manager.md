# Google Tag Manager
Within the **includes** folder of your theme, you will find a `google-tag-manager.php` file. This comes with an example of integrating a Tag manager script into WordPress. In order to take advantage of this you'll just need to uncomment and replace the tag manager script with one provided.

If you aren't sure how to setup a Google Tag Manager account, you can follow [this guide](https://support.google.com/tagmanager/answer/14842164?hl=en).
